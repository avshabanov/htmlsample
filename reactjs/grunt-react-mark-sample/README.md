
## Build

Install everything locally (run once):

```
npm install --save-dev
```

Build app:

```
./node_modules/grunt-cli/bin/grunt
```

Then open build/index.html in the browser

```

### Run tests

```
bash ./scripts/run-tests.sh
```

### Build Single Javascript

```
./node_modules/grunt-cli/bin/grunt
```
